Custom React hooks
useAuth()
Wraps Zustand store. Exposes login, logout, register, current user and loading state.
{ user, login, logout, register, isLoading }
useQuizzes()
Fetches paginated quiz list for current user with loading and error state.
{ data, isLoading, error, refetch }
useCreateQuiz()
Mutation hook. Calls AI generation endpoint, manages all async states.
{ createQuiz, isLoading, error, success }
useQuiz(id)
Fetches single quiz with all questions. Does not expose correct answers.
{ data, isLoading, error }
useAttempt(id)
Fetches attempt state, submitted answers, and full results after completion.
{ data, isLoading, error }
useHistory()
Paginated quiz history with filter and sort controls.
{ data, page, setPage, isLoading }

store/authStore.ts
interface AuthState {
  user: User | null
  accessToken: string | null
  isLoading: boolean
  login: (email, pass) => Promise
  register: (e, p, confirm) => Promise
  logout: () => void
  refreshToken: () => Promise
  loadFromStorage: () => void
}
Axios interceptor flow
1
Every request attaches Authorization: Bearer {token} header automatically
2
On 401 response — silently attempt token refresh using stored refresh token
3
Refresh succeeds — update token in store, retry the original request once
4
Refresh fails — call logout() and redirect to /login
middleware.ts — route guards
Protected/dashboard/* — redirect to /login if no token
Redirected/login and /register — push to /dashboard if logged in
Strategy Read token from cookies for SSR-safe middleware checks
